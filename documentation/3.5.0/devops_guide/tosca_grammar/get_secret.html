get_secret

Edit (pull request)

The get_secret function is used to retrieve secret stored in a Secret Vault technology such as HashiCorp Vault. It can be used anywhere inside Alien Tosca Yaml as a normal function (such as get_property)

Grammar

get_secret: [ secret_path ]

Example

As input of an operation

node_types:
 org.alien4cloud.nodes.SecretComponent:
    interfaces:
      Standard:
        create:
          inputs:
            secret: { get_secret: [ secret/mysql_password ] }
          implementation: scripts/install.sh

As property of a node template

topology_template:
  node_templates:
    SecretComponent:
      type: org.alien4cloud.nodes.SecretComponent
      properties:
        secret_prop: { get_secret: [secret/mysql_password] }
