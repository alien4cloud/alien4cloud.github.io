Administration

Edit (pull request)

Administration section is available to any ADMIN user of the platform. It allows configuration of global elements of the platform including

  • Users
  • Plugins
  • Orchestrators and locations (deployment target)
  • Services allows configuration of services running outside of alien4cloud and services right managements.
  • Artifact repositories allows configuration of external artifact repositories (http, maven, git) from which alien can fetch artifacts to deploy.
  • Server state allows to view the server state and metrics as well as turn on maintenance mode.
  • Audit
  • Meta-properties
