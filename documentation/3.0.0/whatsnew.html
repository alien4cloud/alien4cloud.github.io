New in 3.0.0

Edit (pull request)

    Opensource

    We no longer have a premium distribution for A4C : all previously premium components (plugins, distributions …) have been opensourced.

    Elastisearch upgrade

    Alien4Cloud now supports Elasticsearch version 6.6.2. Data migration guide is available on Github.

    Yorc Orchestrator integration

    This version no longer uses Cloudidy as Orchestrator. We have focused on strongly integrating with Yorc Orchestrator.

    Wizard4Cloud

    This version includes a new simplified UI witch mainly provides a step by step application creation wizard. The following guide describes all of this new UI’s features.

    A4C Spray

    To bootstrap a full A4C stack (including A4C, Yorc, Consul and so on …), a set of Ansible scripts is available on A4C Spray project.
