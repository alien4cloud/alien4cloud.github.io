Scheduler policy

Edit (pull request)

    On the policy view, the allows you to browse policy catalog. Scheduler policy can be found by entering RMS inside the search box :

    • A runtime policy RMSScheduleWorkflowPolicy is defined in the embedded csar.
    • This policy features a schedule time window. Inside this time window, the workflow can be launched if it’s conditions are met. Conditions are expressed using a DSL that admin can enrich.
    • The workflow will be launched when the defined conditions are met.
    • Conditions are initially defined by the administrator and can be modified by the user when preparing the deployment

    • More information is available at RMS Scheduler plugin project.
