    Custom workflows inputs

    Since version 3.0.0, it is possible to specify custom workflow inputs. These inputs can only be defined in topology YAML editor.

    customWorkflowSample:
      inputs:
        comment:
          type: string
          required: true
          description: "Just a string"
        data:
          type: string
          required: true
          constraints:
          - min_length: 5
          description: "A string with a length >= 5"

    You can set the inputs in the custom workflow view details
