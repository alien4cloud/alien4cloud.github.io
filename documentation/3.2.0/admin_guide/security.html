Security

This section describe deprecated manual security settings. We strongly recommend you to automate your installation using A4C Spray project that well manage all SSL aspects for the full A4C stack.

The platform Alien contains multiple components, each component has its own security policy and mechanism. On most of the component, Alien use SSL mutual authentication, or SSL + login.

  • Alien4Cloud UI and Rest API
  • Elasticsearch
