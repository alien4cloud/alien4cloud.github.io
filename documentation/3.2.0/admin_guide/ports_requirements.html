Ports requirements

Edit (pull request)

This section describes all the necessary ports for Alien4Cloud to work. Network traffic must be unrestricted on all of them for the involved servers. Note : Yorc ports are only written here as an indication. If you have any doubt about Yorc required ports, or are using an unmentioned version of Yorc, please check the yorc documentation.

Component - Port description Default port number/range Component Version
Alien4Cloud - standalone GUI port 8088 3.0.X
Yorc 8800 4.0.x
